## 2024-05-24 - Overriding Visible Data with aria-label
**Learning:** Using a static `aria-label` on a button containing dynamic quantitative data (e.g., group item counts) completely overrides the element's text content in the accessibility tree, hiding that data from screen reader users.
**Action:** Always ensure `aria-label` attributes on buttons encompass all meaningful visible text or dynamic counters contained within the button.
